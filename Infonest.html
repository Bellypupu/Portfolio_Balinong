Knowledge Base System

The “Knowledge Base System” project enhances documentation management with a web application that categorizes articles into folders with tags for easy retrieval. To optimize functionality, we utilized Oracle database for its scalability and reliability, managing articles, tags, and metadata securely. Backend operations were powered by C# and Entity Framework, facilitating seamless interactions with the database through object-relational mapping. Python scripts handled data loading, ensuring accurate document imports. This integrated approach accelerates article searches, tracks document changes, and supports CRUD operations, establishing a comprehensive solution for managing and expanding the organization's knowledge repository efficiently.
